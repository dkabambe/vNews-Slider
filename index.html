Initial Release

This is my first publically released JQuery module, as I needed a simple and easy news ticker to fit in with a website I'm developing for a local charity I work with. As I couldn't find what I needed online, I decided to have a go at coding one myself.

I'll admit to be heavily influenced by the design of impressivewebs' Vertical News Slider, as it fitted what I had in mind visually. However, I had a couple of key requirements which led to me coding something myself

  • I wanted a JQuery compatible module to make it easy for me to reuse the slider across my site (and on others!)
  • I needed something which would work with a (Header-News) (Header-News) HTML layout, so that the page would still make logical sense to readers viewing the site without javascript

Simple Installation

Very straightforward to get going - simply include jQuery/jQuery-UI (datepicker), then include this JS file.

You can then customisable use of tags to fit in with site's existing layout. You can also change the speed at which the slider switches to the next item, or turn that off completely.

The code can be as simple as:

HTML: <div id="myID">
  <h4>Title One</h4>
  <div>Content One</div>
  <h4>Title Two</h4>
  <div>Content Two</div>
</div>

JS: $("#myID").vNews();

Hover over here

When you hover over a header item, it will display the relevant content and go back to where you were when you leave.

Hovering over a content item will freeze that content item in place (stop the timer) until you leave.

Options

headerTag: 'h4'

HTML tag which defines the header.

headerWidth: ''

Reserved for future use, (to enable user to define width of header boxes)

dateAttr: 'date'

Atrribute on the header HTML tag which contains timestamp for the news item.

contentTag: 'div'

HTML tag which defines the content.

contentWidth: ''

Reserved for future use, (to enable user to define width of content box)

headerClass: ''

Reserved for future use, (to enable users to apply custom styles to header items)

contentClass: ''

Reserved for future use, (to enable users to apply custom styles to content items)

dateFormat: 'd M yy'

Output format for the date, (using JQuery UI Datepicker formats).

selected: 0

Which news item should be selected on start-up, (esp. useful when ticker set to static via speed: 0)

speed: 5000

Time (in ms) between switching slides. Set to 0 to turn off the ticker and have static display
